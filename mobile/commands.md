# App

## Start App
```
npx expo start
```

## Realizar login no Expo Dev
```
npx expo login
```

## Verificar qual usuário está logado
```
npx expo whoami
```


### Modificar URL da API para execução em Android
```
ios: baseUrl: 'http://localhost:3333'
android: baseUrl: 'http://<IP_MAQUINA>:3333'
```
